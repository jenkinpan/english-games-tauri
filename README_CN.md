# 英语互动游戏中心

> 基于 Tauri v2 + Vue 3 构建的互动英语学习游戏合集，适用于课堂教学与自主练习。

[English](./README.md)

---

## 游戏列表

| 游戏 | 玩法简介 |
|---|---|
| **魔法大富翁** | 最多 4 人本地对战棋盘游戏。投掷 3D 骰子在蛇形棋盘上前进，落格后需答题开宝箱。棋盘内含幸运宝石、神圣护盾、传送门、魔法陷阱、石化诅咒等多种事件格，策略与词汇并重。 |
| **单词炸弹** | 逐张翻牌并朗读英语单词获取积分，但牌堆中隐藏着炸弹——翻到即告本局结束。 |
| **记忆卡片** | 60 秒限时翻牌记忆游戏。单词随机洗牌后正面朝下，在时间耗尽前尽可能多地记住并翻出所有单词。 |
| **单词井字棋** | 双人对战井字棋。每个格子都设有词汇关卡，正确朗读或翻译对应单词才能落子占格。 |
| **女巫的毒药** | 两队各自暗中给一个单词"下毒"，轮流朗读词表——谁念出了对方毒词，谁就输了。 |
| **词汇塔防** | 打字塔防游戏。敌人向城墙持续推进，玩家需根据中文提示在 18 秒内正确拼写英文单词来消灭敌人。 |
| **单词打地鼠** | 中英文单词对从洞中冒出，眼疾手快点击正确答案，准确率直接影响得分。 |
| **谁是幸运儿** | 翻牌寻宝游戏。翻牌时需朗读对应英语单词，找到隐藏的幸运符即获胜。 |
| **看图猜单词** | 输入英语单词逐块揭开遮罩图片，每答对一题还原一块画面，猜出完整图片内容即可通关。 |
| **随机点名** | 课堂点名工具。名字在旋转球体中随机闪动，最终定格选出一位幸运学生。 |
| **单词消消乐** | 双人竞速配对。各自拥有独立的中英文卡片组，先完成全部配对者获胜。 |
| **单词匹配** | 拖拽分类游戏。将散乱的单词卡片拖入正确的分类桶中，考验词汇归类能力。 |

## 技术栈

| 层级 | 技术 |
|---|---|
| 桌面壳 | Tauri v2 |
| 前端框架 | Vue 3 + Vite |
| 开发语言 | TypeScript（严格模式） |
| 样式系统 | Tailwind CSS v4 + Catppuccin 主题 |
| 图标库 | Font Awesome 7 |
| 拼音搜索 | pinyin-match |
| 包管理器 | Bun |

## 环境要求

- [Bun](https://bun.sh/) >= 1.0
- [Rust](https://rustup.rs/) >= 1.77.2
- Tauri v2 系统依赖 — 参考 [Tauri 前置条件指南](https://v2.tauri.app/start/prerequisites/)

---

## 快速开始

```bash
# 安装依赖
bun install

# 启动 Tauri 开发服务器（Rust + Vite 热重载）
bun run dev

# 仅启动前端（无 Tauri，访问 http://localhost:1420）
bun run vite:dev
```

---

## 构建发布包

```bash
# 生产环境构建（输出至 src-tauri/target/release/bundle/）
bun run build
```

支持的桌面打包格式：**macOS**（.dmg + .app）、**Windows**（NSIS 安装包）。

---

## 下载发布版

推送 `v*` 标签后，CI 流水线将自动构建并发布：

- **桌面端**（macOS dmg、Windows NSIS）— `.github/workflows/release.yml`
- **Android APK**（已签名通用包）— `.github/workflows/fast-android-release.yml`

前往 [Releases 页面](https://github.com/jenkinpan/english-games-tauri/releases) 下载最新版本。
